# 控制台-非主流媒体抓取

## 说明

文件名仍保留旧实现名，但当前页面的业务标题是“外媒抓取控制台”。

它管理的是统一外媒收集器，对应：

- `non_mainstream_media_settings`
- `non_mainstream_media_sources`

## 页面能力

- 配置全局抓取频率
- 配置全局抖动
- 按站点启停
- 按 `pipeline_mode` 分组展示

## 分组口径

- `write_flow`：进入编写链路
- `alert_only`：标题提醒链路

当前新增的 4 家媒体会这样出现：

- CoinDesk -> `write_flow`
- Cointelegraph -> `write_flow`
- Decrypt -> `write_flow`
- The Block -> `alert_only`

## 不做的事

- 不重命名数据库表
- 不展示 `mainstream_media` 历史队列
- 不在前端暴露已退役的 `external_media_alert_fetcher`
