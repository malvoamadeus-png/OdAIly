# 编写者2

## 职责

编写者2对应 `x_processing` 的 `format_publish` 阶段，负责：

- 规范标题和正文格式
- 调用发布接口
- 发送内部 Telegram 通知

## 外媒发布约定

- 所有进入编写链路的外媒稿件都保留 `sourceUrl`
- 来源名优先使用 `metadata.site_display_name`
- 路线名统一显示为 `外媒`

这同时适用于：

- `tasks.source = 'non_mainstream_media'`
- 历史兼容 `tasks.source = 'mainstream_media'`

## 仍然隐藏来源的类型

- `blockbeats`
- `panews`
- `jinse`

竞品来源继续不向发布接口透出原始链接。
