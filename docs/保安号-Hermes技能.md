# 保安号 Hermes 技能

> 本技能由 JakeyCha 提供，安装包为 `v1.0.5.zip`。它不属于 OdAIly 主链路，而是值班人员在 Hermes Agent 中使用的 X 控评辅助工具。

## 职责

- 针对 X/Twitter 上造谣或怒骂币安的帖子，生成评论区反击/澄清回复。
- 根据帖子语言自动选择回复语言（当前主要覆盖英语和土耳其语）。
- 非英语回复会附带中文翻译，方便人工挑选最终发布内容。

## 安装前提

- 已安装 [Hermes Agent](https://hermes-agent.nousresearch.com/docs/getting-started/installation)。
- 已从 JakeyCha 获取 `v1.0.5.zip`。

## 安装步骤

### 1. 放置安装包

将 `v1.0.5.zip` 放到仓库 `third-party/保安号/` 目录：

```text
third-party/保安号/v1.0.5.zip
```

### 2. 解压到 Hermes skills 目录

Hermes 的技能统一存放在 `~/.hermes/skills/`。把 zip 解压到该目录即可，无需额外注册。

```bash
./tools/install-hermes-skill-baoanhao.sh
```

或手动执行：

```bash
mkdir -p ~/.hermes/skills
unzip -o third-party/保安号/v1.0.5.zip -d ~/.hermes/skills/
```

### 3. 验证安装

```bash
hermes skills list | grep -i 保安
```

新开 Hermes 会话后，也可在聊天中执行：

```text
/skills
```

确认列表中出现 `保安号`。

如果希望当前会话立即生效，执行 `/reset` 开启新会话。

## 用法

### 反击指定帖子

```text
用保安号反击这个观点：[推特链接]
```

或：

```text
用保安号评论这个观点：[推特链接]
```

把 `[推特链接]` 替换为实际 X/Twitter 帖子 URL。

### 输出说明

- 技能会读取链接对应帖子的语言，并用相同语言生成回复。
- 若原帖为土耳其语等非中文内容，输出会同时给出：
  - 目标语言回复（可直接发布）
  - 中文翻译（便于人工判断）
- 发布前请人工检查，避免 AI 味过重或表述不当。

## 日常巡检流程

当前主要处理英语区和土耳其区针对币安的负面热门帖。建议每几小时或每天至少巡检一次：

1. 用英语或土耳其语在 X 搜索 `Binance` 或 `币安`。
2. 从上到下依次处理负面热门帖，不论流量大小。
3. 对每条目标帖使用保安号生成回复，人工挑选后发布评论。
4. 后期可能会收到统一收集的参考帖单，但仍需人工复核。

目标：保护币安声誉，对造谣和恶意攻击帖进行控评。

## 已知限制

- 输出偶尔仍带有 AI 味，发布前必须人工检查。
- 当前训练重点在英语和土耳其语场景；其他语言效果未充分验证。
- 本技能只生成回复文案，不自动发帖；实际评论仍需人工在 X 发布。
