# 判断者

## 职责

判断者指 `x_processing` 的 `judge` 阶段。

它负责处理：

- `tasks.source = 'x'`
- `tasks.source in ('blockbeats', 'panews', 'jinse')`
- `tasks.source = 'non_mainstream_media'`

## 外媒全文判断

统一外媒全文现在全部通过 `tasks.source = 'non_mainstream_media'` 进入判断者。

判断结果仍只允许：

- `route = 'non_mainstream_media'`
- `route = 'discard'`

这样可以保持现有流水线兼容，但后续写作模板已经统一改成 `mainstream_media_writer`。

## 丢弃规则

外媒全文与其他来源一样，仍会过滤：

- `pure_emotion`
- `baseless_trading_call`
- `daily_chatter`
- `non_crypto_ai`

Decrypt 的非 Crypto 内容主要依赖这里被丢弃。

## 输出

- 保留：`tasks.status = 'judged'`，`x_task_pipeline.news_type = 'non_mainstream_media'`
- 丢弃：`tasks.status = 'discarded'`
