# 收集者-非主流媒体

## 说明

当前实现包名仍为 `non_mainstream_media`，但业务上它已经是统一外媒收集器。

它负责：

- 维护站点注册表
- 发现新文章或新标题
- 按 `pipeline_mode` 决定进入编写链路还是标题提醒链路
- 完成站内去重与 `tasks` 入库

## 站点与模式

### `write_flow`

- `a16z_crypto_posts`
- `forbes_digital_assets`
- `hk01_virtual_assets`
- `coindesk`
- `cointelegraph`
- `decrypt`

这些站点会抓标题、来源链接、正文、作者等完整文章信息，并写入：

- `tasks.source = 'non_mainstream_media'`
- `tasks.status = 'pending'`

### `alert_only`

- `ft_crypto`
- `wsj_business`
- `wsj_economy`
- `wsj_finance`
- `fortune_crypto`
- `the_block`

这些站点只保留标题/摘要级材料，并写入：

- `tasks.source = 'external_media_alert'`
- `tasks.status = 'pending'`

## The Block 特例

- 标题发现使用 `https://www.theblock.co/rss.xml`
- 正文页当前持续受 Cloudflare / 403 阻断
- 因此 The Block 固定停留在 `alert_only`

## 入库元数据

### `write_flow`

- `site_key`
- `site_display_name`
- `pipeline_mode = 'write_flow'`
- `author_names`
- `tags`
- `categories`
- `excerpt`
- `canonical_url`
- `source_kind = 'non_mainstream_media'`

### `alert_only`

- `site_key`
- `site_display_name`
- `pipeline_mode = 'alert_only'`
- `excerpt`
- `source_kind = 'external_media_alert'`
