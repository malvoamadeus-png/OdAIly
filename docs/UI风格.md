UI 风格说明文档（信息流 + 数据平台风格）
1. 整体设计基调

该界面属于典型的**“数据驱动型信息平台 UI”**，融合了金融/资讯/策略类产品的设计语言，整体呈现出：

理性克制：弱装饰、强信息密度
模块清晰：卡片化 + 分栏布局
轻量科技感：通过色彩与微交互体现，而非复杂视觉特效
高可扫描性：强调快速浏览与信息筛选

关键词总结：

信息密度高 / 卡片化 / 浅色主题 / 功能导向 / 低干扰

2. 布局结构（Layout System）
2.1 总体布局

采用典型的三栏结构：

左侧：导航栏（Sidebar）
固定宽度
图标 + 文本垂直排列
功能分区明确（导航 / 工具 / 设置）
中间：主内容区（Main Feed）
信息流 + 卡片列表
强调内容层级（Banner → 分类 → 列表）
右侧：辅助信息区（Aside Panel）
实时信息（快讯）
数据摘要（市场、预测等）
较强“仪表盘”属性

👉 本质是：

「导航 + 内容 + 数据侧栏」的经典信息架构

3. 视觉风格（Visual Language）
3.1 色彩系统
主色调
绿色系（强调上涨/正向）
用于：
数据变化（涨幅）
状态提示
小标签
辅助色
橙色 / 品牌色
用于 Banner 或强调卡片
灰色阶（核心 UI）
背景：#F5F6F7（类似）
卡片：纯白
分割线：极浅灰
风格特点
不依赖渐变（仅少量用于 Banner）
强调“功能色 > 装饰色”
3.2 字体与排版
字体风格
无衬线字体（偏系统字体）
风格：中性、清晰、工具化
字号层级
标题：中等偏大（强调模块）
正文：标准阅读尺寸
辅助信息：较小 + 灰色
排版特点
行距适中，偏紧凑（提高信息密度）
多使用：
单行截断
标签化信息
