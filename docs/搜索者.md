# 搜索者

## 两套搜索阶段

### `x_processing` 搜索阶段

处理：

- `x`
- `blockbeats` / `panews` / `jinse`
- `non_mainstream_media`

统一外媒全文现在也走这条搜索链路，因此 CoinDesk、Cointelegraph、Decrypt 会与 X/竞品一起做：

- Odaily 历史查重
- 运行中 candidate 去重

### `external_media_alert` 搜索阶段

处理：

- `external_media_alert`
- 历史兼容 `mainstream_media`

它只服务标题提醒链路，不负责新的外媒全文写作任务。

## 历史集合

- `odaily_reference_items`
- 本地 search cache
- `external_media_alert_history`

`external_media_alert_history` 现在主要承载标题提醒历史；历史 `mainstream_media` 仍可共用这份窗口。
