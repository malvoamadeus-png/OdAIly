# 监督者

## 职责

监督者负责检查 worker 心跳、队列积压、处理中卡住任务和失败窗口，并发送 Telegram 告警。

## 当前必需心跳

- `x_capture`
- `non_mainstream_media`
- `competitor_monitor`
- `external_media_alert_domain_judge`
- `external_media_alert_search`
- `external_media_alert_notify`
- `x_process_judge_crypto`
- `x_process_judge_ai`
- `x_process_search`
- `x_process_write`
- `x_process_format_publish`
- `x_process_publish`

`external_media_alert_fetcher` 已退役，不再属于必需心跳。

## 任务监控范围

监督者仍覆盖：

- `x`
- `blockbeats`
- `panews`
- `jinse`
- `non_mainstream_media`
- `ai_source`
- `mainstream_media`
- `external_media_alert`
- `ai_source_alert`

其中 `mainstream_media` 仅用于发现历史任务残留或堵塞。

监督者还需要把发布者新增状态纳入告警视野：

- 可领取积压：`publisher_pending`
- 处理中卡住：`publishing`
- 失败状态：`publisher_failed`
