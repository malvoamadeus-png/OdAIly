# 控制台-X抓取

## 职责

控制台-X抓取用于配置收集者-X 的抓取规则，让 worker 启动时加载配置，并在运行中准实时感知变更。

## 配置项

全局配置写入 `x_capture_settings`：

- 全局抓取频率。
- 最大并发数。
- 抖动时间。

账号配置写入 `x_capture_accounts`：

- X 用户名或用户主页 URL。
- 展示名。
- 单账号抓取频率。
- 启用或停用状态。

所有账号默认使用全局频率；单账号配置了频率时，优先生效单账号频率。

## 生效机制

收集者-X：

- 启动时加载 `x_capture_settings` 和 `x_capture_accounts`。
- 运行中监听 `x_capture_config_changed`。
- 收到通知后重新加载内存配置。
- 不需要重启服务。

## 控制台展示

控制台应展示：

- 当前全局配置。
- 账号列表和启停状态。
- 每个账号最近抓取时间、最近成功时间和最近错误。
- 最近 `x_capture_attempts`。
- 最近写入的 X `tasks`。
