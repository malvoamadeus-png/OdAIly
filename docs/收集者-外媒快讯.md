# 收集者-外媒快讯

## 当前状态

这份文档保留给历史兼容链路。

原先独立的主流外媒快讯入口 `external_media_alert_fetcher` 已退役，不再负责新的生产入库。CoinDesk、Cointelegraph、Decrypt、The Block 已并入统一外媒收集器：

- CoinDesk -> `write_flow`
- Cointelegraph -> `write_flow`
- Decrypt -> `write_flow`
- The Block -> `alert_only`

## 历史兼容边界

- `mainstream_media` 任务仍可能存在于历史数据中
- `external_media_alert` 的 `domain_judge/search/notify` 仍兼容处理这些历史任务
- 但新任务不再通过 `media_newsflash -> mainstream_media` 这条路径产生

## The Block 入口

- 官方 RSS：`https://www.theblock.co/rss.xml`
- 列表页：`https://www.theblock.co/latest-crypto-news`
- 正文抓取仍不可用，因此只保留标题提醒用途
