# 收集者-外媒快讯

## 当前状态

这份文档保留给历史兼容链路。

原先独立的主流外媒快讯入口 `external_media_alert_fetcher` 已退役，不再负责新的生产入库。CoinDesk、Cointelegraph、Decrypt、Tether News、The Block 已并入统一外媒收集器：

- CoinDesk -> `write_flow`
- Cointelegraph -> `write_flow`
- Decrypt -> `write_flow`
- Tether News -> `write_flow`
- The Block -> `alert_only`

## 历史兼容边界

- `mainstream_media` 任务仍可能存在于历史数据中
- `external_media_alert` 的 `domain_judge/search/notify` 仍兼容处理这些历史任务
- 但新任务不再通过 `media_newsflash -> mainstream_media` 这条路径产生

## The Block 入口

- 现役统一收集器使用 `Google News RSS` 做标题发现：`https://news.google.com/rss/search?q=site:theblock.co&hl=en-US&gl=US&ceid=US:en`
- `external_media_alert_fetcher` 这条历史入口虽然保留代码兼容，但不再承担生产职责
- The Block 正文抓取仍不可用，因此只保留标题提醒用途
